# Rust Mentor 学习成果库 (Artifacts)

这是存放你 Rust 学习成果的默认目录。

### 🚀 如何激活自动同步？
1. 进入应用的 **设置 (Settings)** 页面。
2. 在 **本地文件同步** 区域，点击“选择您电脑上的文件夹”。
3. 在弹出的对话框中，定位并选中你项目代码库下的这个 `artifacts` 文件夹。
4. 开启 **自动同步** 开关。

### 📂 目录内容
每当你完成一次“费曼解释”或“教练对话”并点击“生成成果”时，系统会自动在这里生成一个 `.md` 格式的知识档案，包含：
- 核心概念总结
- 关键 Rust 代码片段
- 知识漏洞分析 (Gap Analysis)
